## Getting Started

1. Install the dependencies.

**npm:**

```bash
npm i -D eslint @stackup-ui/eslint-config
```

**yarn:**

```bash
yarn add -D eslint @stackup-ui/eslint-config
```

2. Create a `.eslintrc.*` file.

**JSON:**

```json
{
  "extends": ["@stackup-ui/eslint-config"]
}
```

**JS:**

```js
module.exports = {
  extends: ["@stackup-ui/eslint-config"]
}
```
