# Welcome to StackUp UI

StackUp UI is a simple demo design system developed using [Stitches](https://stitches.dev/).

## Installation

### React

**npm:**

```bash
npm i @stackup-ui/react
```

**yarn:**

```bash
yarn add @stackup-ui/react
```
